SMTP / API

Configurations SMTP / API

Laissez vide pour ne pas modifier : clé API (Brevo, etc.), Secret Access Key Amazon SES ou mot de passe SMTP déjà enregistrés.

Serveur recommandé : smtp.office365.com, port 587, chiffrement STARTTLS (TLS — prérempli à l’enregistrement). Utilisateur : adresse e-mail complète du compte Microsoft 365. Mot de passe : compte ou mot de passe d’application si la MFA est activée. L’administrateur doit activer SMTP AUTH (soumission SMTP authentifiée) pour cette boîte ou le locataire. Gouvernement US (GCC High) : utilisez souvent smtp.office365.us comme hôte. Limitez le volume : Microsoft applique des quotas côté locataire.

Pour un compte avec résidence des données en UE, choisissez souvent « Union européenne » pour l’envoi et les appels API. « Automatique » essaie l’UE puis les US uniquement pour l’introspection ; l’envoi Symfony reste sur l’hôte US sauf si vous choisissez l’UE ici.

AWS n’accepte aucun appel API avec seule la clé visible (AKIA…). La Secret Access Key est obligatoire pour lire quotas et identités.

Nouveau template

Fichier TXT : une adresse par ligne. Variables possibles : {{email}} ; si la ligne est au format Prénom/Nom/email@site.com, vous avez aussi {{prenom}}, {{nom}}, {{name}}.

Fichier CSV : après import, mappez les colonnes. Chaque destinataire reçoit les valeurs de sa ligne pour toutes les variables.

  • {{prenom}} / {{first_name}} — prénom
  • {{nom}} / {{last_name}} — nom
  • {{name}}, {{nom_complet}} — nom complet
  • {{email}} — adresse du destinataire
  • {{rotate_url}} ou {{url_rotate}} — URL tirée d’une liste qui tourne (voir ci-dessous)
  • {{date}}, {{time}}, {{datetime}} — date / heure d’envoi
  • Variables aléatoires : {{RANDNUM-6}}, {{RANDALPHA-8}}, {{RANDALPHANUM-10}}
  • Vos colonnes CSV personnalisées : {{ma_variable}} (minuscules et underscores)

Syntaxe acceptée : {{variable}} ou {variable}. Les clés sont insensibles à la casse.

Liens en rotation ({{rotate_url}})

Si vous placez {{rotate_url}} (ou {{url_rotate}}) dans l’objet, le HTML ou le texte, renseignez une URL par ligne. La campagne fera défiler ces liens selon la fréquence ci-dessous (indice global d’e-mails envoyés).

La variable {{rotate_url}} est détectée dans la lettre — pensez à remplir la liste d’URLs.
